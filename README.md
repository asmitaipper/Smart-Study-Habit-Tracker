# Smart Study Habit Tracker

Web app for students to track daily study sessions, visualize patterns, and get smart suggestions based on simple rule-based analysis (ML-style logic).

## Features

- Add study sessions with date, subject, hours, mood, and productivity.
- View all sessions in a clean dashboard-style list.
- Delete any session easily.
- Get suggestions like:
  - When average study hours are too low or too high.
  - When mood is often low or sessions are too long.
  - When you focus too much on one subject.

## Tech Stack

- Node.js + Express (backend API)
- Vanilla HTML, CSS, and JavaScript (frontend)
- In-memory storage for demo (you can later add MongoDB or PostgreSQL)
